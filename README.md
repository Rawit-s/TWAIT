# 잡설
TWAIT는 로이스의 첫번째 프로젝트로, 프로그램 지식이 없이도 팩파일에 대한 번역작업을 할 수 있도록 프로그램만 간단하게 실행하면 모드 번역을 할 수 있는 자동화 프로그램입니다.



# 사용법
1. remove_non_text.exe 다운로드
2. 실행
3. 팩파일을이 있는 폴더를 지정하면 text 폴더를 제외한 모든 폴더를 제거합니다 (백업필수!)
